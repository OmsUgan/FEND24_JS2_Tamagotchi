Välj ett husdjur

Inga husdjur än!

Börja med att adoptera ett husdjur genom formuläret till vänster. Du kan adoptera upp till 4 olika husdjur och ta hand om dem!

Tips för att ta hand om ditt husdjur:

  1. Använd knappen "Lek" för att göra ditt husdjur glad

  2. Använd knappen "Ät" för att mata ditt husdjur

  3. Använd knappen "Vila" för att återställa energi

  4. Låt inte någon statistik nå noll, annars kommer ditt husdjur att rymma!
